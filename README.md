A personal repository of independent data analysis and coding projects in economics. Each project applies quantitative methods and Python-based (possibly R-based in the future) tools to explore economic questions, patterns, and datasets.

This repository serves as a portfolio of self-directed economics research projects. Work here is independent and covers a range of topics including macroeconomics, international economics, and more. Each project folder is self-contained and includes its own data, code, and write-up.
